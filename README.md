# 🚀 Portafolio Profesional de Desarrollador Web

Este proyecto es un portafolio personal desarrollado como parte de un trabajo práctico para demostrar habilidades en **HTML5**, **CSS3** y el uso del framework **Tailwind CSS**.

---

## 🌐 Demo en Vivo
Puedes ver el sitio funcionando en el siguiente enlace:
👉 [https://m-robledo.github.io/mi-portafolio/](https://m-robledo.github.io/mi-portafolio/)

---

## 🛠️ Tecnologías y Herramientas
* **HTML5:** Estructura semántica para mejor SEO y accesibilidad.
* **Tailwind CSS:** Framework de utilidades para un diseño moderno y responsive.
* **Google Fonts:** Uso de la fuente "Inter" para una tipografía limpia.
* **Formspree:** Integración funcional para el formulario de contacto.
* **Git & GitHub:** Control de versiones y despliegue.

## 📋 Características
- [x] **Diseño Responsive:** Adaptable a móviles, tablets y escritorio (Flexbox/Grid).
- [x] **Modo Oscuro:** Botón funcional para cambiar el tema de la página.
- [x] **Multi-página:** Navegación fluida entre Inicio y Contacto.
- [x] **Formulario Real:** Recibo de mensajes directamente por correo electrónico.
